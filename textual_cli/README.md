# Textual CLI

A simple command-line interface built with the Python Textual framework.

## Features

- `/hello` command: Displays a greeting message
- `/exit` command: Exits the application

## Requirements

- Python 3.7+
- Textual library

## Installation

```bash
pip install textual
```

## Usage

Run the application:

```bash
python cli_app.py
```

Once the application is running:
1. Type `/hello` and press Enter to see a greeting
2. Type `/exit` and press Enter to exit the application

## Project Structure

- `cli_app.py`: Main application file
